Minecraft

This is a website with the link and instrustions on how to play minecraft 1.8 at school / home.

This version of minecraft is coded in javasScript in stead of normal java, which allows it to be run in a web browser. This website is also hosted on github so it is not able to be blocked by school admins (kind of)

Instrustions

To start of click the link labeled " Download " then when the page loads in the top right corner click the download button. Once downloaded double click the file and it will take you to a 'website'. follow the instruton of the website and you should have a fully fleged version up and running in your web browser.
